# Contributing to OpenGifs

Thank you for your interest! Here's how to help:

## Reporting Issues

- Search existing issues before opening a new one
- Include steps to reproduce, expected vs actual behavior
- Mention your PHP version, database, and hosting environment

## Pull Requests

1. Fork and clone the repository
2. Create a branch: `git checkout -b feature/my-feature`
3. Make your changes
4. Run `composer run-script` if any linters are configured
5. Commit with a clear message
6. Push and open a PR

## Code Style

- Follow PSR-12 coding standards
- Keep English as the only language in source code and comments
- Test upload and search flows manually after changes
